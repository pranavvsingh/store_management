Toggle Price Display

Some stores may not want to show their prices, such as in the case of a wholesale supplier, or perhaps you want the customer to call for pricing on certain items. This document leads you through the steps necessary to turn off the price display in all pages and modules included in a standard installation. It is all done via CSS, so the only modifications needed to your PHP code is the addition of a CSS class in the various presentation sections. No changes are made to any SQL queries or business functions.

Here is the list of files you will need to modify. As always, it is highly recommended you make a backup of these files before making these changes. Jump to each instruction set by clicking on the list item below:

  • stylesheet.css
  • includes/boxes/shopping_cart.php
  • shopping_cart.php
  • product_info.php
  • includes/boxes/specials.php
  • includes/boxes/whats_new.php
  • checkout_confirmation.php
  • products_new.php
  • product_reviews.php
  • products_reviews_info.php
  • product_reviews_write.php

To hide prices in your store, you first need to create a class in the stylesheet that handles this:

Open up stylesheet.css and add the following class: <Back to Top> <Next file>

.hidePrices {
	display:none;
}

This allows you to quickly turn them all back on if you change your mind later. Alternatively, you could creat a separate class for each module, which would allow you to turn price displays on or off individually.

To hide prices in the shopping cart:

In the file includes/boxes/shopping_cart.php, find the following code block starting at around line 58:<Back to Top> <Next file>

 if ($cart->count_contents() > 0) {
   $info_box_contents[] = array('text' => tep_draw_separator());
   $info_box_contents[] = array('align' => 'right',
                                'text' => $currencies->format($cart->show_total()));
 }

Add the "hidePrices" class by wrapping it around the word "text" as shown below:

 if ($cart->count_contents() > 0) {
   $info_box_contents[] = array('text' => tep_draw_separator());
   $info_box_contents[] = array('align' => 'right',
               '<span class="hidePrices">;text</span>' => $currencies->format($cart->show_total()));
 }

In the file shopping_cart.php find the following text at around line 71:<Back to Top> <Next file>

    $info_box_contents[0][] = array('align' => 'right',
                                    'params' => 'class="productListing-heading"',
                                    'text' => TABLE_HEADING_TOTAL);

Do the same thing to the word 'text' as shown below:

    $info_box_contents[0][] = array('align' => 'right',
                                    'params' => 'class="productListing-heading"',
          '<span class="hidePrices">text</span>' => TABLE_HEADING_TOTAL);

This will remove the Price column from the display.

And I believe that does it. This same technique can be applied to any contributions or custom modules and pages you may have designed. By leaving the queries and logic that return the prices alone, and only hiding the display of the prices, you assure that reverting back to a full price display is achieved simply and cleanly, by commenting out or deleting the display:none attribute of the hidePrices class in stylesheet.css.
